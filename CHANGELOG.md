# История разработки алгоритма (Changelog)

Развитие проекта от простых скриптов парсинга до гибридного нейросетевого конвейера.

## [1.1.0] - 2026-03-14
### Добавлено
- **Open Source:** Архитектура и математика проверок официально опубликована в открытом доступе на GitHub для подтверждения экспертности (E-E-A-T).
- **LLM Интеграция:** В финальный отчет добавлена оценка грамматики и прямой вывод вероятности использования ИИ-генераторов.

## [1.0.0] - 2026-02-12
### Релиз
- **Public Launch:** Официальный запуск веб-интерфейса Antiplagiat.best.
- **Безопасность:** Жёстко внедрена и задокументирована политика Zero-Retention. Все вычисления переведены в оперативную память (RAM) без создания промежуточных логов проверяемых текстов.

## [0.8.0] - 2025-11-28
### Изменено
- **Слияние Pipeline:** Разрозненные скрипты объединены в единый конвейер. Оценка шинглов, векторное сравнение и анализ "воды" теперь выдают единый взвешенный скорринг уникальности. 
- Переработана математика пенальти: точный копипаст штрафуется сильнее (коэффициент 1.0), чем синонимический рерайт (0.8).

## [0.5.0] - 2025-09-10
### Добавлено
- **Фокус на ИИ (Смена парадигмы):** Стало очевидно, что классический рерайт отмирает, уступая место ChatGPT. Начата разработка AI Scanner.
- Внедрен анализ Бурстовости (ритм предложений) и Перплексии для выявления стерильного машинного "пластика".

## [0.3.0] - 2025-05-14
### Изменено
- **Внедрение Эмбеддингов:** Шинглы перестали справляться с умными синонимайзерами. Проведены первые успешные тесты использования косинусного сходства (Cosine Similarity) векторов для поиска украденных смыслов.

## [0.1.0] - 2025-01-25
### Инициализация
- Заложен фундамент проекта. Написан первый прототип парсера на базе метода N-грамм (шинглов). 
- Длина шингла стабилизирована на значении 4 слов для баланса между точностью и количеством ложных срабатываний (False Positives) на юридических терминах.
